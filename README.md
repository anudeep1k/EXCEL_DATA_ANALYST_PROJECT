# 🚴 Bike Sales Dashboard in Excel

## 📌 Project Overview
This project focuses on building an **interactive Bike Sales Dashboard in Microsoft Excel** to analyze customer purchasing behavior and identify the key factors influencing bike sales.

Starting from a **raw customer dataset**, the data was cleaned, transformed, and converted into a structured format to support analysis. Using **Pivot Tables, Pivot Charts, and Slicers**, I created a dynamic dashboard that allows users to explore trends across demographics, income, commute distance, education, and occupation.

The goal of this project is to demonstrate how **Excel can be used as a business intelligence tool** to convert raw data into actionable insights for decision-making.

---

## 📂 Dataset Information
The dataset includes customer-level bike sales information with the following fields:

- ID
- Marital Status
- Gender
- Income
- Children
- Education
- Occupation
- Home Owner
- Cars
- Commute Distance
- Region
- Age
- Purchased Bike

### 🔹 Feature Engineering
A new column called **Age Brackets** was created to improve segmentation:
- Adolescent: 0–30
- Middle age: 31–54
- Old: 55+

## 🔍 Key Business Insights
- Customers aged **31–54** had the highest bike purchase rate
- Customers with **5–10 mile commute distance** showed stronger buying behavior
- Income and gender influenced purchasing decisions
- Education and occupation revealed clear customer segments
- Interactive slicers enable dynamic business exploration
